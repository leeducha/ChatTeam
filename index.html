🔥 Kết nối Firebase

Để mọi người thật sự thấy nhau từ các máy tính khác nhau, bạn cần kết nối Firebase Realtime Database (miễn phí).

1

Truy cập console.firebase.google.com → Tạo project mới → Vào Realtime Database → Tạo database

2

Vào Rules tab → Paste: { "rules": { ".read": true, ".write": true } } → Publish

3

Vào Project Settings → Your apps → Thêm Web App → Copy firebaseConfig object

💡 Lưu ý: Firebase Realtime Database miễn phí cho đến 1GB dữ liệu và 10GB/tháng bandwidth. Đủ dùng cho nhóm lớp!
